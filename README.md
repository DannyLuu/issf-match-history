# ISSF Match History

Private match log for ISSF-style pistol events.

This repository records:
- Match names
- Match dates
- Event type
- Scores
- Location
- Notes
- Target and score-sheet images

## Events Tracked

- 25m Standard Pistol
- 25m Sport Pistol
- Other ISSF / club-level pistol events

## Master Results

See [`matches.csv`](matches.csv).

## Match Reports

Individual match reports are stored in the [`matches/`](matches/) folder.

## Images

Images are stored by match date and event in the [`images/`](images/) folder.
