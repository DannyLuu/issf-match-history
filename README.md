## About This Repository

**Included:**
- Match names, dates, and locations
- Scores and percentages
- Placement and classification (when available)
- Score breakdowns and evidence links

**Excluded:**
- Training notes and practice analysis
- Firearm, ammunition, or equipment details
- Personal technical notes

## How to Use

- **Master Index:** See [`matches.csv`](matches.csv) for the complete results table
- **Match Details:** Click on any report above for full details, score breakdowns, and evidence
- **Template:** Use [`matches/template.md`](matches/template.md) when adding new matches

## Repository Structure

```
issf-match-history/
├── README.md (this file)
├── matches.csv (master results index)
├── matches/
│   ├── template.md (use for new entries)
│   └── [individual match reports]
└── Evidence/
    └── [dated evidence folders]
```

## Adding New Matches

1. Add evidence files under `Evidence/YYYY-MM-DD Match Name - Event/`
2. Update [`matches.csv`](matches.csv)
3. Create a detailed report in `matches/` using the template

## Result Status Reference

| Status | Meaning |
|---|---|
| **Pending** | Result entered but not yet checked |
| **Unofficial** | Recorded but not yet verified |
| **Verified** | Confirmed against score sheet or evidence |
| **Published** | Appears on official result page |
| **Corrected** | Updated after a correction |
